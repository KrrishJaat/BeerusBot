# ⚡ BeerusBot — Telegram Moderation Bot

> A powerful modular Telegram bot for moderation, automation, and GitHub-based ROM management.

---

## 🚀 Features

- 🔥 Moderation (Mute, Warn, Ban, Hakai)
- 👥 Group management (Welcome, Rules, Logs)
- 📦 Notes, AFK, Filters
- ⚙️ Auto tasks (JobQueue)
- 🔗 GitHub workflow integration (ROM builds)

---

## ⚙️ Setup

### 1. Clone
- git clone https://github.com/yourusername/BeerusBot.git
- cd BeerusBot

### 2. Install
pip install -r requirements.txt

### 3. Create .env
- BOT_TOKEN=your_bot_token
- OWNER_ID=your_id
- GITHUB_TOKEN=your_token

### 4. Run
python bot.py

## 📸 Commands

| Command | Description |
|--------|------------|
| /start | Start bot |
| /mute | Mute user |
| /unmute | Unmute user |
| /warn | Warn user |
| /unwarn | UnWarn user |
| /dwarn | Delete Message + Warn user |
| /resetwarn |Reset All Warn of user |
| /ban | Ban user |
| /unban | Unban user |
| /notes | Manage notes |
| /afk | AFK system |
| /hakai | Global Ban |
| /unhakai | UnGlobal Ban |
| /convert | Convert Currency |
| /checkfw | Check Device's Latest Fimware (Samsung Only) |
| And Many More... |
